T1: 洛谷题面

这道题看起来是很简单的题

方法一：暴力俗话说，暴力出奇迹，由于部分分很足，暴力足以得 90pts

90pts code 这是 Lattle_car 大佬看错限制的初次代码

#include <bits/stdc++.h>
using namespace std;
int a,b,c;
int ansa,ansb,ansc;
int n;
int maxsum=-1,maxmin=-1;
bool flag=1;
int main()
{
  freopen("order.in","r",stdin);
  freopen("order.out","w",stdout);
  cin>>n;
  if(n<3)
  {
    cout<<-1;
    return 0;
  }
  for(a=0; a<=n/7; a++)
    for(b=0; b<=n/4; b++)
      for(c=0; c<=n/3; c++)
      {
        int sum=a+b+c;
        int minn=min(a,min(b,c));
        int kkk=7*a+4*b+3*c;
        if(kkk==n)
        {
          flag=0;
          if(minn>maxmin)
          {
            ansa=a;
            ansb=b;
            ansc=c;
            maxsum=sum;
            maxmin=minn;
          }
          else if(minn==maxmin&&maxsum<sum)
          {
            ansa=a;
            ansb=b;
            ansc=c;
            maxsum=sum;
            maxmin=minn;
          }
        }
      }
  if(flag) cout<<-1;
  else cout<<ansa<<' '<<ansb<<' '<<ansc;
  return 0;
}

正解：打表俗话又说，数学上来先打表

我们可以发现一个规律：除了1,2,51,2,51,2,5其他都能找出正确方法，然后又发现能找出方法的，每加14，最优解的各个方案数都加上1，如：

n 3 17 31 ...
ans 0,0,1 1,1,2 2,2,3 ...
n 10 24 38 ...
ans 0,1,2 1,2,3 2,3,4 ...
n 14 28 42 ...
ans 1,1,1 2,2,2 3,3,3 ...

.........

然后我们只要在一些不成立的数特判一下就好了。
